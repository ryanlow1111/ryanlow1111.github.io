Project 1
Accommodation Booking App

Overview

This application is built by React Native CLI. User able to search accommodation and book the accommodation. Firestore is used as the backend of this application.

Main screen

The application will start on this screen.

User able to use search bar which located in top screen to search accommodation. Bottom-tab navigation is used to navigate between main screen and profile screen.

Location search

User will direct to this screen after tap search bar from main screen. User able to search the location they wanted.

Guests screen

User choose the amount of guests in this screen. User cannot enter -1 amount of guests.

List of Location

The list of accommodation is stored in firestore and is render with flatlist.

Details of accommodation

Accommodation details will get from firestore and displayed in this screen.

User able to tap the 'Book' button to make booking.

Profile

Once user tap on the 'Book' button, an alert will shown if the booking is succesful.

Booking details will store into the firebase.

Login

User able to login thier account. AsyncStorage is used to store user login in the local storage.

Profile

Profile page will display the user name and their booking list.

User able to tap the accommodation they booked to view the details of the accommodation

Accommodation details from booking List

Accommodation deatils after user tap the accommodation in booking list.

User can cancel the booking by tap 'Cancel Book' button in the botttom of the screen.

Cancel Booking

An alert will shown if cancel booking is succesful.

Booking details will be removed from firestore.
